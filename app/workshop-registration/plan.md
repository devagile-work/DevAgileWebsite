Day 1: The Building Blocks (HTML, CSS, & JS Crash Course)
Session (2 Hours):

0-30 mins: Intro to how the web works (Client vs. Server). The DOM. Basic HTML structure.

30-60 mins: CSS basics (Flexbox is critical here. Skip grids for now).

60-120 mins: JavaScript essentials: variables (let/const), functions (arrow functions), arrays, objects, and basic DOM manipulation.

Take-Home Task (1-2 Hours): Build a static HTML/CSS profile card. Style it nicely. Add a JS button that toggles "dark mode" on the card.

Day 2: React Fundamentals
Session (2 Hours):

0-30 mins: Why React? Introduce JSX and components. Use Vite to set up a project.

30-90 mins: Props and State (useState). How data flows downward.

90-120 mins: Handling events (clicks, form inputs).

Take-Home Task: Build the shell of Project 1 (Portfolio). Create the components for the Header, Footer, and a Projects section. Hardcode the project data.

Day 3: Advanced React & API Fetching
Session (2 Hours):

0-60 mins: The useEffect hook. Why we need it and the dependency array dependency array (this trips everyone up).

60-120 mins: Fetching data from a public API (e.g., a dummy JSON API) and rendering it in a list. Introduce React Router for multi-page feel.

Take-Home Task: Finish Project 1. Add a "Blog" or "Articles" page that fetches fake posts from jsonplaceholder.typicode.com and displays them.

Day 4: Backend Basics (Node.js & Express)
Session (2 Hours):

0-30 mins: What is a server? Intro to Node.js and npm.

30-90 mins: Setting up an Express server. Understanding routes (GET, POST).

90-120 mins: Testing API endpoints using Postman or Thunder Client.

Take-Home Task: Set up the Express server for Project 2 (Task Master). Create hardcoded GET and POST routes that just return JSON messages (e.g., { message: "Task added" }).

Day 5: Connecting Frontend to Backend
Session (2 Hours):

0-60 mins: CORS (what it is and how to fix it). Making fetch requests from the React app to the local Express server.

60-120 mins: Handling asynchronous data on the frontend while waiting for the backend to respond (loading states).

Take-Home Task: Connect the React frontend of Project 2 to the Express backend. The frontend should display whatever hardcoded data the backend sends.

Day 6: Databases (MongoDB & Mongoose)
Session (2 Hours):

0-30 mins: Intro to NoSQL databases. Setting up a free MongoDB Atlas cluster.

30-90 mins: Connecting Express to MongoDB using Mongoose. Defining Schemas and Models.

90-120 mins: Replacing the hardcoded backend routes with actual database operations (Create and Read).

Take-Home Task: Finish the CRUD operations for Project 2. The React app should now be able to add a task, save it to the database, and read it back. Add the "Delete" functionality.

Day 7: Deployment & Wrap Up
Session (2 Hours):

0-60 mins: Deploying the backend to a free service (e.g., Render, Railway). Managing environment variables (keeping database passwords secret).

60-90 mins: Deploying the frontend to Vercel or Netlify. Connecting the deployed frontend to the deployed backend.

90-120 mins: Workshop wrap-up. How to write good README files for their GitHub repos so recruiters notice them.

Take-Home Task: Polish both projects. Add proper README files with screenshots. Add both projects to their resumes!
